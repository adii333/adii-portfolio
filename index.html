Financial Market App – UI/UX Case Study

Mobile App Figma UI/UX Designer Practice Project

Overview

A mobile app that displays real-time stock prices, personal portfolio data, and news headlines – all in one dashboard.

Problem Statement

Users struggle to track live prices and portfolio performance in a simple interface.

Goal

Create a clean, user-friendly design that includes:

  • Real-time stocks
  • Portfolio stats
  • Trends & charts
  • News feed

Design Process

  • Research: Analyzed 3 finance apps
  • Wireframing: Basic sketches
  • UI Design: High-fidelity in Figma
  • Testing: Focused on dark theme, readability, and layout

Reflection

Improved layout skills, component consistency, and finance app UX understanding.
